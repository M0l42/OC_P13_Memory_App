The Memory App

About the project

Jeune développeur et fan du contenu de Fabien Olicard, j'ai décidé de réadapté la Mémory Box en version application web pour mon projet final de ma formation chez OpenClassroom. Etant curieux d'apprendre de nouvelle chose tout les jours, notamment de nouvelle langues, cette application est faite pour permettre à tout ce qui veulent d'apprendre de nouvelles informations.

La méthode

Les flashcards, c'est quoi ? Ce sont de petites fiches de révisions simplifiées avec, pour chacune, sa question et sa réponse, optimisées pour le fonctionnement du cerveau. Associées aux méthodes d'apprentissage de la Memory box, elles vous aideront à mémoriser rapidement, facilement et de manière ludique toutes les informations que vous avez toujours voulu retenir !
